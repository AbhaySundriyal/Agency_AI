This is the frontend of the Agency AI platform, designed to provide a smooth, responsive, and modern user interface.

Tech Stack:

React for building dynamic UI components

Tailwind CSS for styling and responsive layouts

Axios for API integration

Features:

Responsive dashboard for managing AI-driven services

Interactive components and cards

Integration-ready with backend APIs

Modern, clean design for agency workflows

This repo is intended to work alongside the backend API (separate repo) to deliver a full-stack Agency AI solution.
